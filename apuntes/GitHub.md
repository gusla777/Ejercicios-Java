# Apuntes de Github

## Que es de github?

Es un google drive con herramientas

## Repositorios

 Espacio para alojar codigo como si fuera una carpeta.Normalmente se usa una repo para cada proyecto

## Como Entro

1. Por la web
2. Por la app
3. Por comandos

## Comandos

Siempre antes de irnos:

- ***COMMIT***:crear un checkpoint.
- ***PUSH***:Guardar todos los cambios en los servidores de github (subir cambios a la nube).

Al empezar:

- ***FETCH***: Comprueba los cambios que ha habido desde el ultimo Fetch.
- ***PULL***:Descarga los cambios.
